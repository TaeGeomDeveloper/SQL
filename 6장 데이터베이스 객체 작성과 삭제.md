# 26강 DDL 테이블 작성, 삭제, 변경

### CREATE TABLE
```MySQL
테이블 작성
CREATE TABLE sample (
  no INTEGER NOT NULL,
  a VARCHAR(30),
  b DATE);
```

### DROP TABLE 
```MySQL
DROP TABLE sample;
```

### ALTER TABLE
```MySQL
열 추가
ALTER TABLE sample ADD newNum INTEGER;
```
```MySQL
열 속성 변경
ALTER TABLE sample MODIFY newNum VARCHAR(20);
```
```MySQL
열 삭제
ALTER TABLE sample DROP no;
```
